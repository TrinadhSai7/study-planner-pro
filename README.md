Smart Study Planner

Project Overview
Smart Study Planner is a web application that helps students plan their studies efficiently.
It generates a timetable based on subjects and deadlines, tracks progress, and gives simple suggestions to improve study habits.

Features
Add subjects with deadlines and difficulty
Generate study timetable automatically
Mark tasks as completed
Track progress with percentage
Get suggestions for better study planning

Tech Stack
Frontend HTML CSS JavaScript
Backend Python Flask
Database SQLite

How to Run the Project
Install Python
Install Flask using pip install flask
Run the application using python app.py
Open browser and go to http://127.0.0.1:5000/

Project Structure
smart-study-planner folder
app.py
templates folder with index.html
static folder with style.css and script.js
database.db

How It Works
User enters subjects and deadlines
System creates a timetable
User marks tasks as completed
Application tracks progress and gives suggestions
